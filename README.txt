# Projetos Flutter

Este repositório reúne diversos projetos desenvolvidos em Flutter, cada um em sua própria pasta. Aqui você encontrará exemplos práticos de aplicações mobile para Android e iOS, com foco em:

- **Boilerplates** prontos para iniciar novos apps  
- **Demonstrações** de widgets e layouts (ListView, GridView, Formulários)  
- **Integrações** com APIs REST, SQLite e Firebase  
- **Recursos avançados** como animações, geolocalização e notificações push  

## Estrutura

/projeto_exemplo_1/ → App de lista de tarefas
/projeto_exemplo_2/ → Chat em tempo real com Firebase
/projeto_exemplo_3/ → App de mapas e geolocalização

Cada pasta contém:
1. Código-fonte Flutter (`lib/`, `assets/`, etc.)  
2. `pubspec.yaml` configurado com dependências  
3. README específico com instruções de execução  

## Como usar

1. Clone este repositório:  
   ```bash
   git clone https://github.com/seu-usuario/flutter-projects.git
   cd flutter-projects

2. Entre na pasta do projeto desejado:
  cd projeto_exemplo_1

3. Instale as dependências e execute:
  flutter pub get
  flutter run
